# yt-dlp-dearpygui

A minimal YouTube downloader built with **Python**, **Dear PyGui**, and **yt-dlp**.

<img width="597" height="246" alt="d" src="https://github.com/user-attachments/assets/36f8d681-3d59-4eb8-9fd7-dafdc36773fd" />

## Features

* Analyze a YouTube video before downloading
* Display video title
* Show uploader name
* Show video duration
* Show like count
* Show comment count
* Download the best available video quality
* Download audio only
* Real-time download progress bar
* Simple and lightweight desktop interface

## Planned Features

* Display video thumbnail
* Play a notification sound when a download finishes

## Requirements

* Python 3.10+
* `yt-dlp`
* `dearpygui`

Install the dependencies:

```bash
pip install yt-dlp dearpygui
```

Downloaded files are saved in the `SavedFiles/` directory.
